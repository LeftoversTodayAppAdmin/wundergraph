# WunderGraph Typesafe Golang Client

#### Getting started

```shell
npm install && npm start
```

#### Get all Continents

```shell
cd go && go mod tidy && go run main.go
```

## Test the API

1. (Optional) Copy the `.env.example` file to `.env.test` and fill in the required values.
2. Run the following command:

```shell
npm run test
```
